Inline Styles are #1 in the Hierarchy

(*Paragraph element)(*In index.html: p style="color: green" id="title">Inline Styles are #1 in the Hierarchy) (*Inline style selectors rule over all the others so the line is green) (*The line would be green except for the '!important' rule in style.css below)

The '!Important' rule will override ALL previous styling rules in all the Selected Elements

(*Paragraph element)(*In style.css: p { color: palevioletred !important; })

Nested Selector Specificity

(*In index.html: h2 class="box" id="box">Nested Selector Specificity, is 'nested' within a: div class="container")(*In style.css: #box { color: red; } & .container #box {color: chocolate;} & .container .box { color: aqua; } & .container h2 { color: chartreuse; })(*color: chocolate; wins the specificity hierarchy)(*If the nested selectors in style.css were unnested, the color would be red!)
